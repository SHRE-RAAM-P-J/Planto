# 🌿 Planto - Modern Plant Landing Page

A modern responsive plant ecommerce landing page built using React, Vite and Tailwind CSS.

##  Features

- Fully Responsive Design
- Modern Glassmorphism UI
- Reusable React Components
- Tailwind CSS Styling
- Smooth Hover Effects
- Clean Layout Structure

##  Tech Stack

- React
- Vite
- Tailwind CSS
- React Icons

##  Installation

```bash
npm install
npm run dev
```

##  Deployment

Deployed using Vercel.

##  Folder Structure

```bash
src/
 ├── components/
 ├── assets/
 ├── App.jsx
 ├── main.jsx
 └── index.css
```
